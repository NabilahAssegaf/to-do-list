# Drag and Drop To-Do List

This project is a simple drag-and-drop to-do list application built with HTML, CSS, and JavaScript. It allows users to create, manage, and reorder their tasks easily.

## Project Structure

```
drag-drop-todo
├── src
│   ├── index.html          # Main HTML document for the application
│   ├── css
│   │   └── styles.css      # Styles for the application
│   ├── js
│   │   ├── app.js          # Main JavaScript file for functionality
│   │   └── drag-drop.js     # Logic for drag-and-drop functionality
│   └── components
│       └── todo-item.js    # Class representing a single to-do item
├── package.json            # npm configuration file
├── .gitignore              # Files and directories to ignore by Git
└── README.md               # Documentation for the project
```

## Features

- Add new to-do items
- Mark items as complete
- Delete items
- Drag and drop to reorder items

## Getting Started

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd drag-drop-todo
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the application.

## Usage

- To add a new task, type in the input field and press Enter.
- To mark a task as complete, click on the task.
- To delete a task, click the delete button next to it.
- To reorder tasks, click and drag them to your desired position.
